CSS - Cascate Style Sheet

O CSS, ou folha de estilo em cascata, é usado para formatar quaisquer itens de um documento. ele pode ser aplicado de duas formas:

  1. Inline -> usado como propriedade do elemento HTML.
  2. Por referencia -> dentro de uma folha de estilo.

Formatacão realizada inline. Ou seja, diretamente no elemento HTML.
